# SOC Detection Lab

ELK-based SIEM lab simulating MITRE ATT&CK techniques using Atomic Red Team.

## Purpose
To validate telemetry, analyze attacker behavior, and develop SOC-level detection skills.

## Architecture
Windows VM (Sysmon + Winlogbeat) → Elasticsearch → Kibana

## Techniques Covered
- T1016 – System Network Configuration Discovery
